## 🧠 Key Concepts Explored

### 1. Data Handling & Preprocessing
*   **Data Cleaning:** Identifying and handling missing values, duplicates, and outliers using Pandas.
*   **Feature Engineering:** Transforming raw data into formats suitable for machine learning models.
*   **Exploratory Data Analysis (EDA):** Using visualizations to understand data distributions and correlations before modeling.

### 2. Machine Learning Fundamentals
*   **Data Splitting:** Understanding the importance of separating data into Training and Testing sets to prevent overfitting.
*   **Model Training & Inference:** Fitting classical models (e.g., Linear Regression, Random Forest) using Scikit-Learn.
*   **Evaluation Metrics:** Moving beyond "accuracy" to understand Precision, Recall, MSE, etc., based on the problem context.

### 3. Modern AI Applications (LLMs & Vision)
*   **API Integration:** Interacting with pre-trained models (like OpenAI APIs) to understand prompt engineering and request handling.
*   **Computer Vision Basics:** Loading, transforming, and interpreting image data for object detection or classification tasks.
